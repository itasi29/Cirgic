【開発情報】
・環境：DxLib
・言語：C++
・期間：2ヶ月(200時間)
・人数：1人

【操作方法】
A：決定・アクセル
B：キャンセル・バック
RB：ドリフト※1
LB：アイテム使用
START：オプション
左スティック：カーソル移動・ハンドル操作

※1ドリフトについて
　この作品のドリフトは一度ふわっとジャンプし、着地してからドリフトに遷移します。
　一定時間(エフェクトの色がオレンジから青に変わる時)経ってから離すと加速します。

【アイテム】
・コイン　　：所持枚数に応じて速度が変わります(最大10枚まで所持可能)
・ダッシュ　：一度だけ前方にダッシュすることができます。
・ダッシュx3：3回前方にダッシュすることができます。使い切るまでは新しいアイテムは取得できません。

【レース】
・3ラップで1レースです。
・途中にあるゲート(ミニマップ上で赤い線がある場所)を順番に通らないとラップが進みません
